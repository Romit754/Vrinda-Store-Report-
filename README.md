📌 Project Overview
This project analyzes sales data from Vrinda Store using Microsoft Excel. The goal is to extract meaningful insights from raw transactional data and build an interactive dashboard to support business decision-making.

🎯 Objectives
Analyze overall sales performance 📈
Understand customer demographics (age & gender) 👥
Identify top-performing states 🗺️
Evaluate sales channels performance 🛒
Track order status distribution 📦


📂 Dataset Description

The dataset contains multiple sheets:

Vrinda Store → Raw transactional data
Order Vs Sales → Monthly sales & order trends
Men Vs Women → Gender-wise sales analysis
Order Status → Order distribution (Delivered, Cancelled, Refunded)
State Sales → State-wise revenue contribution
Age & Number → Age group & gender analysis
Channel → Sales by platforms (Amazon, Flipkart, Ajio, etc.)
Annual Report → Dashboard summary


📊 Key Insights
💰 Sales Trends
Monthly sales show consistent growth patterns.
Peak sales observed during specific months (seasonal trends).

👨‍🦱👩‍🦰 Gender Analysis
Women contribute significantly higher sales compared to men.
Major revenue driver: Female customers.

📦 Order Status
Majority of orders are successfully delivered.
Small percentage of cancellations and refunds.

🌍 State-wise Performance
Top-performing states include:
West Bengal
Andhra Pradesh
Haryana
